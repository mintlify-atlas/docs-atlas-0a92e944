## What is FriendNet?

FriendNet is a self-hosted file sharing system inspired by Soulseek, designed specifically for small groups of friends. It uses a central server to facilitate file search and peer discovery while keeping file downloads peer-to-peer.

Unlike traditional P2P systems, FriendNet provides **optional proxied downloads**, allowing users behind restrictive firewalls or VPNs to share files without complicated port forwarding setup.

## How It Works

FriendNet uses a three-component architecture:

1. **Server** - Central coordinator that handles authentication, search indexing, and proxy connections
2. **Desktop Client** - Runs on your computer to share files and includes a built-in web UI
3. **QUIC Protocol** - Custom protocol over QUIC for secure, fast peer-to-peer communication

When direct peer-to-peer connections aren't possible, the server automatically proxies file transfers, ensuring everyone can participate regardless of their network setup.
